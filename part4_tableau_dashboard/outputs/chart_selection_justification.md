# Chart Selection Justification

## Sales Trend View
- Question Answered: How are sales changing over time?
- Chart Type: Line Chart
- Reason: Line charts are ideal for showing trends across time periods.
- Fields Used:
  - X-axis: Order Date
  - Y-axis: Sales
- Design Principle Applied: Clear trend visibility.
- Mistake Avoided: Avoided pie charts for time-series data.

---

## Regional Performance View
- Question Answered: Which region performs best?
- Chart Type: Bar Chart
- Reason: Bar charts make comparisons across regions easy.
- Fields Used:
  - Region
  - Sales
  - Profit for color encoding
- Design Principle Applied: Easy comparison through aligned bars.
- Mistake Avoided: Avoided cluttered geographic maps.

---

## Category Profitability View
- Question Answered: Which categories generate the highest profit?
- Chart Type: Bar Chart
- Reason: Allows direct comparison of category profitability.
- Fields Used:
  - Sub Category
  - Profit
  - Category as color
- Design Principle Applied: Clear ranking and sorting.
- Mistake Avoided: Avoided 3D charts.

---

## Customer Segment View
- Question Answered: Which customer segment contributes most to sales?
- Chart Type: Bar Chart
- Reason: Enables straightforward comparison.
- Fields Used:
  - Segment
  - Sales
  - Profit Margin as color
- Design Principle Applied: Consistent color usage.
- Mistake Avoided: Avoided unnecessary labels.

---

## Shipping Performance View
- Question Answered: Which shipping mode causes delays?
- Chart Type: Bar Chart
- Reason: Average delivery time comparison is simple to understand.
- Fields Used:
  - Ship Mode
  - Average Delivery Days
- Design Principle Applied: Minimal clutter.
- Mistake Avoided: Avoided overly complex visualizations.

---

## Discount vs Profit View
- Question Answered: How does discount affect profitability?
- Chart Type: Scatter Plot
- Reason: Scatter plots reveal relationships between variables.
- Fields Used:
  - Discount
  - Profit
- Design Principle Applied: Visual correlation analysis.
- Mistake Avoided: Avoided aggregated charts that hide patterns.
