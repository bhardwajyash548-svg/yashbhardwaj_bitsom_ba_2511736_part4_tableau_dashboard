# Retail Executive Dashboard

## Business Problem
The leadership team requires a dashboard to monitor sales performance, profitability, customer behavior, shipping efficiency, discount impact, and return patterns to support business decisions.

## Dataset Description
The dataset contains retail sales transactions including:
- Sales
- Profit
- Discount
- Region
- Customer Segment
- Category
- Shipping Mode
- Order Date

## Tableau Workbook
The dashboard was created using Tableau and saved as:

tableau/executive_dashboard.twbx

## Calculated Fields Created
1. Profit Margin
2. Cost
3. Average Order Value
4. Return Rate
5. Shipping Delay Bucket

## Dashboard Components
- Sales Trend
- Regional Performance
- Category Profitability
- Customer Segment Analysis
- Shipping Performance
- Discount vs Profit Analysis
- Return Analysis
- KPI Cards

## Filters Used
- Region
- Category
- Customer Segment
- Date
- Ship Mode

## Interactions Used
Selecting a region filters all related charts in the dashboard.

## Key Business Insights
- South region leads in sales.
- Technology category drives profit.
- Discounts negatively affect profitability.
- Standard shipping experiences delays.

## Dashboard Story Summary
The dashboard highlights strengths, weaknesses, opportunities, and risks to support leadership decision-making.

## Assumptions and Limitations
- External factors were not considered.
- Customer lifetime value data was unavailable.
- Competitor data was unavailable.
